# [RU] Star Wars: Starting Systems and Portrait Collection Pack

Неофициальная русская локализация мода **Star Wars: Starting Systems and Portrait Collection Pack**.

- Оригинальный мод: https://steamcommunity.com/sharedfiles/filedetails/?id=1708335879
- Автор оригинала: force728
- Поддерживаемая версия Stellaris: 4.4.x
- Версия локализации: 1.0.0

## Установка и порядок загрузки

Для работы требуется оригинальный мод. В игровом наборе разместите локализацию **после оригинала**.

Локализация не содержит игровых ресурсов оригинала и не изменяет его файлы.

## Известное ограничение

Несколько фраз в событиях Куата, подсказках Звёздной кузницы и один шаблон флота записаны в оригинальном моде как текст с пробелами, а не как допустимые ключи локализации. Они остаются на английском, чтобы русификатор не переопределял игровые скрипты и не ломал совместимость с обновлениями оригинала.
